About

         The World Wide Web (abbreviated as WWW or W3[2] and commonly known as the Web) is a system of interlinked hypertext documents accessed via the Internet. With a web browser, one can view web pages that may contain text, images, videos, and other multimedia and navigate between them via hyperlinks. The Internet is a global system of interconnected computer networks. In contrast, the Web is one of the services that runs on the Internet. The computer receiving the HTTP request delivers it to Web server software listening for requests on port 80. If the web server can fulfill the request it sends an HTTP response back to the browser indicating success

          Computer Science VII Semester has two labs according to VTU syllabus. The subject code of networks lab is 06CSL77 and web lab is 06CSL78. They are web programming lab and networks lab. Web programming lab includes programs on JavaScript, XHTML, CSS, Perl, PHP and so on. These programs gives example of how a web page is created and how they are executed in the browser. The networks lab deals with simulating different types of traffic in the network using different protocols and LAN connection. There are some programs which also illustrates with algorithms which comes across the networking protocols.

          A computer network, often simply referred to as a network, is a collection of hardware components and computers interconnected by communications channels that allow sharing of resources and information. Networks may be classified according to a wide variety of characteristics such as the medium used to transport the data, communications protocol used, scale, topology, and organizational scope. The World Wide Web had a number of differences from other hypertext systems that were then available. The Web required only unidirectional links rather than bidirectional ones. This made it possible for someone to link to another resource without action by the owner of that resource.
